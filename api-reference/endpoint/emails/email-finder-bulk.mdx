# Use Case:
Start an asynchronous bulk job to find verified work emails for many leads.

# Pricing
**$0.02** per valid email at Tier 0 (base price).

You are charged only when a valid, verified email is successfully found. If no email can be determined or the result is invalid, the request is free.

<Tip>Your price per result decreases automatically as your lifetime spend grows. [Check your current tier and pricing →](https://beta.generect.com/settings/billing)</Tip>
