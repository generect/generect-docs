## Use Case:
Retrieve a company profile from cached data by Generect ID, LinkedIn URL, domain, or company name.

## Pricing
This endpoint consumes credits per returned result. If no company is found, no credit is charged. See [Pricing](/billing/pay-as-you-go) for credit costs.
