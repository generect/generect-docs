## Use Case:
Retrieve a full lead profile from cached data by Generect ID, LinkedIn URL, or email (reverse lookup).

## Pricing
**$0.02** per result at Tier 0 (base price).

Each database enrichment request returns a profile from Generect's cached data. You are charged only when the requested company or lead is found — if no data is returned, the request is free.

<Tip>Your price per request decreases automatically as your lifetime spend grows. [Check your current tier and pricing →](https://beta.generect.com/settings/billing)</Tip>
