## Use Case:
Retrieve a full lead profile from cached data by Generect ID, LinkedIn URL, or email (reverse lookup).

## Pricing
This endpoint consumes credits per returned result. If no lead is found, no credit is charged. See [Pricing](/billing/pay-as-you-go) for credit costs.
