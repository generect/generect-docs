## Use Case:
Retrieve a fresh company profile by Generect ID, LinkedIn URL, domain, or company name.

## Pricing
**$0.04** per result at Tier 0 (base price).

Each real-time enrichment request queries live data sources (LinkedIn, Crunchbase, websites) to return a fresh profile. You are charged only when the requested company or lead is found — if no data is returned, the request is free.

<Tip>Your price per request decreases automatically as your lifetime spend grows. [Check your current tier and pricing →](https://beta.generect.com/settings/billing)</Tip>
