# Use Case:
Start an asynchronous bulk phone finder job.

# Pricing
**$0.40** per found phone at Tier 0 (base price).

You are charged only when a valid phone number is successfully found. If no phone number can be determined, the request is free.

<Tip>Your price per result decreases automatically as your lifetime spend grows. [Check your current tier and pricing →](https://beta.generect.com/settings/billing)</Tip>
