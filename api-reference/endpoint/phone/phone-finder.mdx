# Use Case:
Find a phone number by `lead_id`, `linkedin_url`, or by `first_name` + `last_name` + `company`.

# Pricing
**$0.40** per found phone at Tier 0 (base price). Charged only when a valid phone is found.

<Tip>Pricing decreases as your lifetime spend grows — [see all tiers](/billing/managing-billing#spend--save-tiers-pricing-model).</Tip>
