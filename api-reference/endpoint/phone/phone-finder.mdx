## Use Case:
Find a phone number by `lead_id`, `linkedin_url`, or by `first_name` + `last_name` + `company`.

## Pricing
This endpoint consumes credits per returned result. If no phone is found, no credit is charged. See [Pricing](/billing/pay-as-you-go) for credit costs.
