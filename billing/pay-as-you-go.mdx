Generect uses a transparent Pay-as-you-go model — you only pay for what you actually use.

No subscriptions, no hidden fees — just real consumption.

## Core principles of the model

1. **You pay only for results.**  
   For example, Email Finder charges only for a *valid* found email, and enrichment charges only when data is found.
2. **No monthly plans.**  
   You're not locked into subscriptions — use the platform at your own pace.
3. **Pricing decreases as you grow.**  
   The higher your lifetime spend, the better your pricing tier becomes, and the cheaper each future action costs.
4. **A positive balance is required.**  
   Email Finder, API usage, exporting data, and email validation all require a positive balance.  
   If your balance reaches zero, you won't be able to use these features until you top up.

## Typical Charges

Below is a quick overview of common API charges.

| API action | What you pay for | Credits charged |
| ----- | ----- | ----- |
| **Preview leads** `POST /preview/leads/` | Masked lead preview result | **Free** for $2000+/month accounts; otherwise **0.001 credit per result** |
| **Preview lead count** `POST /preview/leads/count/` | Counting matching preview leads | **Free** |
| **Database search** `/search/database/*` | Cached search result | **0.1 credit per result** |
| **Database search count** `/search/database/*/count/` | Counting cached matches | **Free** |
| **Realtime search** `/search/realtime/*` | Fresh search result | **1 credit per result** |
| **Realtime search count** `/search/realtime/*/count/` | Counting fresh matches | **1 credit per count request** |
| **Lead enrichment** `/enrich/*/lead/` | Full lead profile returned | Charged only when a lead is found |
| **Company enrichment** `/enrich/*/company/` | Full company profile returned | Charged only when a company is found |
| **Email Finder** `POST /email/find/` | Valid email found by `lead_id` or name + domain | **1 credit per valid email** |
| **Bulk Email Finder** `POST /email/find/bulk/` | Valid email found in a bulk job | **1 credit per valid email** |
| **Email Validation** `POST /email/validate/` | Email validation request | **1 credit per validation** |
| **Account endpoints** `/accounts/*` | Profile, usage, and transaction lookup | **Free** |

<Note>
  No Data, No Charge: if an email, lead, or company is not found, Generect charges **0 credits** for that result.
</Note>

<Tip>
  The more you use Generect, the better your pricing becomes. Check your current tier in the Billing section to see your discounts.
</Tip>
