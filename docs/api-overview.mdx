The Generect API allows you to integrate the platform's capabilities directly into your products, internal systems, or automated workflows.  
In simple terms, it's a way to work with Generect data without manual actions in the UI, at scale and with full cost control.

The API is designed for companies that want to:

- automate lead and company search
- display data inside their own products
- enrich CRM systems with up-to-date information
- pay only for real value, not for "empty" results

## Key Principles of the Generect API

### Free preview → pay only for revealed data

You can show search results for free and pay only when you decide to reveal an email, phone number, or full profile.

### Database or Realtime — depending on your task

Use fast and cost-efficient data for large-scale workflows, or ultra-fresh data for precise outreach.

### No Data, No Charge

If data is not found, no credits are charged.

## Core API Capabilities and Business Scenarios

### Preview Search for Contacts and Companies

**What this gives you**  
The Preview API lets you find relevant contacts and companies and display basic information without revealing sensitive data (email, phone, LinkedIn URL).

You can:

- find contacts and companies by role, industry, and location
- display name, job title, company, and industry
- browse results without paying
- reveal detailed data only when it's truly needed

**Ideal for:**

- SaaS platforms
- internal CRM or admin panels
- AI SDR and automation tools
- preview → reveal business models

### ICP-Based Search and Segmentation

**What this gives you**  
The Generect API allows you to automatically build lists of companies and contacts based on your Ideal Customer Profile.

You can segment by:

- industry
- company size
- location
- roles and seniority

The API supports two modes:

**Database Mode — fast and cost-efficient**  
Best for large lists and recurring workflows.

**Realtime Mode — maximum data freshness**  
Best for targeted searches and time-sensitive signals.

### Contact and Company Enrichment

**What this gives you**  
If you already have partial or outdated data, the API allows you to enrich and refresh contact and company profiles.

You can:

- keep your CRM up to date
- enrich profiles before outreach
- work with LinkedIn URLs, domains, or internal IDs

Available options:

- fast enrichment from cache (Database)
- ultra-fresh data retrieval (Realtime)

### Email Finding and Validation

**What this gives you**  
The API allows you to:

- find corporate email addresses
- validate deliverability
- reduce bounce rate and sender reputation risks

You only pay if a valid email is found.
